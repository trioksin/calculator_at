Консольный калькулятор на Python

Консольный калькулятор, написанный на Python, поддерживающий основные арифметические операции и сохранение истории вычислений в текстовый файл.


Программа позволяет:

 Складывать числа
 Вычитать числа
 Умножать числа
 Делить числа
 Просматривать историю всех вычислений
 Автоматически сохранять каждую успешную операцию в файл history.txt
 Обрабатывать ошибки ввода
 Защищаться от деления на ноль
 Структура проекта
calculator/
│
├── main.py          # Основной файл программы
├── history.txt      # История вычислений (создается автоматически)
└── README.md        # Документация проекта

Запуск программы
1. Клонировать репозиторий

Перейдите в папку проекта

2. Запустить программу
python main.py


После запуска пользователь увидит меню:

Калькулятор

1. Сложение
2. Вычитание
3. Умножение
4. Деление
5. История
0. Выход

Далее необходимо выбрать нужный пункт меню.

Например:

Выбери действие: 1
Введите первое число: 10
Введите второе число: 25
Результат: 35
История вычислений

Все успешно выполненные операции автоматически сохраняются в файл

history.txt

Пример содержимого файла:

5.0 + 10.0 = 15.0
18.0 - 4.0 = 14.0
9.0 * 6.0 = 54.0
25.0 / 5.0 = 5.0

Просмотреть историю можно, выбрав пункт меню:

5. История
Как работает программа
Сохранение истории

Для записи операций используется функция:

def save_history(text):
    with open(history_file, "a", encoding="utf-8") as file:
        file.write(text + "\n")

Она открывает файл в режиме добавления (a) и записывает новую операцию, не удаляя предыдущие записи.

Просмотр истории

История отображается функцией:

def show_history():

Если файл истории отсутствует, программа сообщает пользователю:

История пока пуста.
Главное меню

Работа программы организована через бесконечный цикл

while True:

Это позволяет выполнять неограниченное количество операций до тех пор, пока пользователь не выберет пункт

Поддерживаемые операции

Сложение
Формула:
a + b
Пример:
15 + 20 = 35

Вычитание
Формула:
a - b
Пример:
20 - 8 = 12

Умножение
Формула:
a * b
Пример:
6 * 9 = 54

Деление
Формула:
a / b
Перед выполнением операции программа проверяет, что второе число не равно нулю.
Если пользователь введет: 0
то увидит сообщение
Ошибка: деление на ноль!
__________________________________________________________________________________________________________________________________________________________

Обработка ошибок

Программа использует конструкцию
try:
для безопасного ввода данных.
Если пользователь введет текст вместо числа:
Введите первое число:
например: Привет
то программа сообщит:
Ошибка: нужно вводить числа!

и продолжит работу.
